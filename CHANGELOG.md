# 更新日志 (CHANGELOG)

## [1.1.0] - 2024-03-13

### ✨ 新功能

#### 博客风格首页重新设计
- **Hero 区域**: 添加渐变背景、大标题、副标题和主要操作按钮
- **栏目展示**: 工作空间以卡片式网格布局展示,类似博客栏目
- **文章列表**: 公开文档以博客文章形式展示,包含:
  - 文章标题和最后更新时间
  - 内容摘要(自动提取前 200 字)
  - 预计阅读时长(基于 400 字/分钟)
  - "阅读全文"链接
- **微交互动画**: 卡片悬停效果、平滑过渡、视觉反馈
- **响应式设计**: 完美适配移动端、平板和桌面端

### 🎨 设计改进
- 采用紫色渐变主题色 (`#667eea` → `#764ba2`)
- 优化字体层级和间距系统
- 添加 SVG 图标增强视觉识别
- 统一动画缓动函数 `cubic-bezier(0.4, 0, 0.2, 1)`

### 🌐 国际化
新增翻译键:
- `home.subtitle` - Hero 副标题
- `home.getStarted` - "开始使用"按钮
- `home.exploreCategories` - 栏目区说明
- `home.recentPosts` - "最新文章"标题
- `home.latestUpdates` - 文章区说明
- `home.readMore` - "阅读全文"链接
- `home.readTime` - 阅读时长显示
- `home.poweredBy` - "Powered by"
- `home.noPublicContentHint` - 空状态提示

### 📚 文档
- 新增 `docs/blog-homepage-design.md` - 详细设计文档
- 新增 `docs/blog-homepage-guide.md` - 使用指南
- 更新 `CLAUDE.md` - 添加最新功能说明

### 🔧 技术改进
- 优化 CSS 样式组织,新增 500+ 行博客风格样式
- 完善 TypeScript 类型定义
- 通过 TypeScript 类型检查
- 前端构建成功,文件大小优化

- Markdown 实时协作编辑器

- Git 风格版本快照
- 标题级别细粒度权限
- 工作空间管理
- 附件上传和管理
- 用户认证和权限系统

- 浏览器扩展(文章导入)

**版本号规范**: 遵循语义化版本 (Semantic Versioning)
