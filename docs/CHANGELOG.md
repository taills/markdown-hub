# 更新日志 (CHANGELOG)

## [1.3.0] - 2026-03-24

### 🔧 重构

#### UI 组件全面迁移至 Preline UI
- **全面移除自定义 CSS**: `web/src/styles.css` 从 20KB+ 缩减至 247字节，所有样式迁移至 Preline UI 组件和 Tailwind 工具类
- **Preline 组件覆盖**: Tree View、Modal/Overlay、Toast、Accordion、Button、Input 等
- **Tailwind 原子化**: 所有布局、间距、色彩、字体使用 Tailwind 工具类
- **深色模式完善**: 所有组件支持 Tailwind `dark:` 模式变体
- **MarkdownPreview 重构**: HTML 生成函数直接输出带 Tailwind 类的内容

#### 专业 UI 设计系统
- **设计令牌**: 完整的 CSS 变量系统（色彩、阴影、过渡、圆角）
- **专业组件样式**: 导航面板、图标按钮、用户头像、状态指示器、表格、卡片等
- **动画与交互**: 平滑过渡动画、脉冲状态指示、滑动通知
- **空状态设计**: 统一的空状态组件，包含图标、标题和描述
- **深色模式**: 完整的暗色主题支持，使用 Slate 色系

### 📚 文档更新
- `DESIGN.md` 更新至 v1.3，新增 Preline UI 实现说明
- `CLAUDE.md` 更新前端技术栈，新增 Preline UI 使用规范

- **标题锚点评论**: 针对特定标题位置的评论

- **编辑删除**: 作者可编辑删除，管理员可删除任何评论
- **权限控制**: 基于文档权限的评论访问控制

#### AI 助手功能
- **智能问答**: 基于文档内容的 "Ask AI" 功能
- **文档总结**: AI 自动总结文档要点
- **文档补全**: AI 辅助补全文档内容
- **内容扩写**: AI 扩写选定的文档内容
- **对话历史**: 支持多轮对话上下文

## [1.1.0] - 2024-03-13

### ✨ 新功能

#### 博客风格首页重新设计
- **Hero 区域**: 添加渐变背景、大标题、副标题和主要操作按钮
- **栏目展示**: 工作空间以卡片式网格布局展示,类似博客栏目
- **文章列表**: 公开文档以博客文章形式展示,包含:
  - 文章标题和最后更新时间
  - 内容摘要(自动提取前 200 字)
  - 预计阅读时长(基于 400 字/分钟)
  - "阅读全文"链接
- **微交互动画**: 卡片悬停效果、平滑过渡、视觉反馈
- **响应式设计**: 完美适配移动端、平板和桌面端

### 🎨 设计改进

- **主题色更新**: 从橙色系(`#f59e0b`)统一为紫色渐变系(`#667eea` → `#764ba2`)

- **按钮样式**: 所有主按钮改为紫色渐变背景 + 白色文字 + 悬停上浮动画
- **交互反馈**: 统一 Hover/Focus/Active 状态的紫色反馈
- **阴影系统**: 统一使用紫色系阴影,增强视觉层次
- **更新范围**:
  - 登录/注册页面
  - 文档列表和工作空间
  - 编辑器和工具栏
  - 侧边栏面板(快照、权限、附件)
  - 个人中心
  - 管理后台(用户管理、站点设置)
  - Markdown 预览(链接、引用块)
  - 加载动画和公共组件
