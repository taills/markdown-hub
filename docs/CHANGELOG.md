# 更新日志 (CHANGELOG)

## [1.1.0] - 2024-03-13

### ✨ 新功能

#### 博客风格首页重新设计
- **Hero 区域**: 添加渐变背景、大标题、副标题和主要操作按钮
- **栏目展示**: 工作空间以卡片式网格布局展示,类似博客栏目
- **文章列表**: 公开文档以博客文章形式展示,包含:
  - 文章标题和最后更新时间
  - 内容摘要(自动提取前 200 字)
  - 预计阅读时长(基于 400 字/分钟)
  - "阅读全文"链接
- **微交互动画**: 卡片悬停效果、平滑过渡、视觉反馈
- **响应式设计**: 完美适配移动端、平板和桌面端

### 🎨 设计改进

#### 全站配色统一更新
- **主题色更新**: 从橙色系(`#f59e0b`)统一为紫色渐变系(`#667eea` → `#764ba2`)
- **CSS 变量系统**: 新增 `--primary`, `--primary-dark`, `--primary-gradient` 等主题变量
- **按钮样式**: 所有主按钮改为紫色渐变背景 + 白色文字 + 悬停上浮动画
- **交互反馈**: 统一 Hover/Focus/Active 状态的紫色反馈
- **阴影系统**: 统一使用紫色系阴影,增强视觉层次
- **更新范围**:
  - 登录/注册页面
  - 文档列表和工作空间
  - 编辑器和工具栏
  - 侧边栏面板(快照、权限、附件)
  - 个人中心
  - 管理后台(用户管理、站点设置)
  - Markdown 预览(链接、引用块)
  - 加载动画和公共组件

#### 视觉一致性
- 采用紫色渐变主题色 (`#667eea` → `#764ba2`)
- 优化字体层级和间距系统
- 添加 SVG 图标增强视觉识别
- 统一动画缓动函数 `cubic-bezier(0.4, 0, 0.2, 1)`
- 统一 Focus 阴影效果 `0 0 0 3px rgba(102, 126, 234, 0.15)`

### 🌐 国际化
