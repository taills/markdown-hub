# 全站配色统一更新文档

## 📋 更新概述

将整个 MarkdownHub 前端站点的配色方案统一为博客首页的紫色渐变主题(`#667eea` → `#764ba2`),替换原有的橙色系(`#f59e0b`),实现全站视觉风格的一致性和现代化。

## 🎨 配色方案更新

### CSS 变量系统

```css
:root {
  /* 主题色 - 紫色渐变系 */
  --primary: #667eea;           /* 主色调 */
  --primary-dark: #764ba2;      /* 深紫色 */
  --primary-light: #8b9bef;     /* 浅紫色 */
  --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

  /* 功能色 */
  --accent: #667eea;            /* 强调色(使用主题紫色) */
  --success: #16a34a;           /* 成功(绿色 - 保持不变) */
  --danger: #ef4444;            /* 危险(红色 - 保持不变) */
  --warn: #f97316;              /* 警告(橙色 - 保持不变) */

  /* 辅助色 */
  --hover-overlay: rgba(102, 126, 234, 0.1);
  --active-overlay: rgba(102, 126, 234, 0.2);
}
```

## 🎯 设计原则

### 1. 颜色使用规范
- **主按钮**: 使用紫色渐变,白色文字
- **次要按钮**: 透明背景,灰色边框,Hover 变紫色
- **危险操作**: 保持红色
- **警告操作**: 保持橙色
- **成功状态**: 保持绿色
- **链接/强调**: 使用主题紫色

### 2. 交互反馈
- **Hover**: 边框变紫色 + 紫色背景叠加层(10%透明度)
- **Focus**: 紫色边框 + 3px 紫色外阴影(15%透明度)
- **Active**: 紫色背景叠加层(20%透明度)
- **按钮 Hover**: 上浮 1-2px + 阴影增强

### 3. 动画效果
- **缓动函数**: `cubic-bezier(0.4, 0, 0.2, 1)` (Material Design)
- **过渡时间**:
  - 快速反馈: 0.2s (Hover, Focus)
  - 中等动画: 0.3s (按钮,弹窗)
  - 慢速动画: 0.5s (页面切换)

### 4. 阴影系统
```css
/* 轻度阴影 */
box-shadow: 0 2px 8px rgba(102, 126, 234, 0.3);

/* 中度阴影(Hover) */
box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);

/* Focus 阴影 */
box-shadow: 0 0 0 3px rgba(102, 126, 234, 0.15);
```

## 📝 使用建议

### 如何添加新按钮
```css
/* 主按钮 */
.my-button {
  background: var(--primary-gradient);
  border: none;
  border-radius: 6px;
  padding: 0.6rem 1rem;
  color: white;
  font-weight: 600;
  cursor: pointer;
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
  box-shadow: 0 2px 8px rgba(102, 126, 234, 0.3);
}

.my-button:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
}

.my-button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  transform: none;
}
```

### 如何添加 Focus 效果
```css
.my-input:focus {
  outline: none;
  border-color: var(--primary);
  box-shadow: 0 0 0 3px rgba(102, 126, 234, 0.15);
}
```

### 如何添加 Hover 效果
```css
.my-element:hover {
  border-color: var(--primary);
  color: var(--primary);
  background: var(--hover-overlay);
}
```
